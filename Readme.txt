1. create "client" folder for front-end

Client folder:
1. run npx create-react-app in client
2. delete node modules, package-lock.json and yarn install to replace it with yarn
3. yarn add react-router-dom, reactstrap, react-bootstrap, axios, react-icons
4. yarn start
5. start with localhost:3000 to login
